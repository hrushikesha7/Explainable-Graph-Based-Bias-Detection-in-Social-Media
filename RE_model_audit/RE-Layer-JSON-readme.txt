1. metrics_global.json: Contains the single numbers (Fairness Gap, Accuracy, Mean Risks).
    Used for: Big summary cards at the top of the dashboard.

2. data_features.json: Contains the Feature Names and their Importance Scores.
    Used for: Interactive Bar Charts (hover to see values).

3. data_polarization.json: Contains the list of risk scores for Group A and Group B.
    Used for: Dynamic Histograms or Density Plots.

4. data_network.json: Contains the Nodes and Edges for the Case Study user.
    Used for: Interactive Network Graph (drag and drop nodes).

5. data_recommendations.json: Provides a ranked list of the top 10 items (users) recommended to the target user.
